This plugin helps you obtain and publish Google Sidewiki comments in your WordPress blog.

== Description ==

= With WP oySidewiki you can: =

= English: =
* Automatically get all Google Sidewiki comment for each post or page URI in your blog.
* ... and display this comments in your template!

= Español: =
* Obtiene automáticamente los comentarios de Google Sidewiki para cada URI en tu blog.
* ... y los muestra integrados en tu template!

= What is WP oySidewiki? =
= English: =
WP oySidewiki is a simple plugin to integrate native comments with Google Sidewiki comments.

Enjoy your feedback in your site !

= Español: =
WP oySidewiki es un simple plugin que integra los comentarios de Google Sidewiki en tu blog.

Disfruta de tu feedback en tu sitio !

== Installation ==
= English: =
1. Upload `wp-oysidewiki` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo getSidewikiComments(get_permalink()); ?>` in your template
4. Enjoy !

= Español: =
1. Subir la carpeta `wp-oysidewiki` en `/wp-content/plugins/`
2. Activar el plugin en el menu 'Plugins' de WordPress
3. Agrega `<?php echo getSidewikiComments(get_permalink()); ?>` en tu template
4. Disfrutalo !

= Uninstall =

= English: =
1. If you wish to remove the plugin completely, press desactivate. (No cache and no database needed)

= Español: =
1. Si quieres remover el plugin, presiona desactivar. (No utiliza cache ni base de datos)

== Frequently Asked Questions ==

= WP oySidewiki use cache files ? =

No. No cache, and no database needed. 
= WP oySidewiki adds many data in my DB? =

Noup. This plugins uses all existing post data URI you have been entered through Wordpress Editor, and get Google Sidewiki comments from Google.

= Can I customize the tag for better control of my layout? =

Yes. You can change CSS design for comments.
